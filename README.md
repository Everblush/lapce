## Installation

### Manual
- ```mkdir ~/.local/share/lapce-stable/Everblush```
- copy ```Everblush.toml``` and ```volt.toml``` into ```~/.local/share/lapce-stable/Everblush```
- ```cp *.toml ~/.local/share/lapce-stable/Everblush/```
